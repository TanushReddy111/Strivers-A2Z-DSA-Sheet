# Striver's A2Z DSA Sheet — Python Solutions

A structured collection of solutions to problems from [Striver's A2Z DSA Course/Sheet](https://takeuforward.org/strivers-a2z-dsa-course/strivers-a2z-dsa-course-sheet-2/), implemented in Python. This repository documents my journey through Data Structures and Algorithms, with each problem accompanied by its statement and my solution. 

---

## About

This repository serves as both a personal learning log and a reference resource. Each problem is given its own folder containing the problem statement and a Python solution, making it easy to revisit concepts and track progress over time.

The goal is consistency: working through the sheet end-to-end while writing reinforcing core patterns that appear across technical interviews.

---

## Repository Structure

Each problem follows a consistent layout:

```
Problem Name/
├── problem.txt    # Problem statement with Examples
└── sol.py         # Python solution
```

Top-level view:

```
Strivers-A2Z-DSA-Sheet/
├── Check if a number is Armstrong Number or not/
├── Check if a number is Palindrome or Not/
├── Count digits in a number/
├── Find GCD of two numbers/
├── Print 1 to N using Recursion/
├── Print Name N times using Recursion/
├── Reverse Digits of A Number/
└── ...
```

## How to Use

1. Navigate to any problem folder.
2. Read `problem.txt` for the problem statement.
3. Open `sol.py` to view the implementation.

To run a solution locally:

```bash
python "Folder Name/sol.py"
```
